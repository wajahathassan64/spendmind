# SpendMind Privacy Policy

This repository hosts the privacy policy for the SpendMind iOS application.

## Privacy Policy

The privacy policy is available at:
**https://wajahathassan64.github.io/spendmind/privacy-policy.html**

## Setup

This repository uses GitHub Pages to host the privacy policy. The HTML file is served directly from the repository root.

## Updating

To update the privacy policy:
1. Edit `privacy-policy.html`
2. Commit and push changes
3. Changes will be live automatically via GitHub Pages
